# Streamlit AI Chatbot

A minimalist, interactive AI chatbot interface built with Streamlit and powered by the OpenAI API.

## Overview
This project provides a sleek, browser-based chat interface. It demonstrates how to integrate modern AI language models into a user-friendly web application with minimal boilerplate code.

## Features
- **Interactive UI:** Real-time chat interface powered by Streamlit.
- **AI Integration:** Seamless connection to OpenAI's language models.
- **Session Management:** Maintains conversation history for a better user experience.
- **Deployment-Ready:** Designed for quick deployment on Streamlit Cloud.

## Getting Started

### Clone the repository:
```bash
git clone https://github.com/cansatir/streamlit-ai-chatbot.git
```
### Run the application:

```Bash
streamlit run app.py
```
## Learning Objectives
- Building interactive web apps with Streamlit.

- Working with OpenAI API endpoints.

- Understanding the architecture of LLM-based applications.
